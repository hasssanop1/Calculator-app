🧮 Calculator Application

A simple and user-friendly Calculator Application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. This project is perfect for beginners who want to practice logic building and improve their programming skills.

🚀 Features

➕ Addition

➖ Subtraction

✖ Multiplication

➗ Division

🧹 Clear / Reset functionality

📱 Responsive and easy-to-use interface (if web-based)

HTML – Structure of the calculator
